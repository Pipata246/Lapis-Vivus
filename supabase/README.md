# Supabase — миграции

Все изменения схемы БД хранятся в `supabase/migrations/` как SQL-файлы.

## Как применить миграцию

1. Открой [Supabase Dashboard](https://supabase.com/dashboard) → свой проект.
2. **SQL Editor** → New query.
3. Скопируй содержимое нужного файла из `migrations/` (например `001_create_users.sql`) и выполни (**Run**).

## Порядок миграций

| Файл | Описание |
|------|----------|
| `001_create_users.sql` | Таблица `users` для пользователей бота |

Новые миграции добавляй с номером `002_...`, `003_...` и т.д.
